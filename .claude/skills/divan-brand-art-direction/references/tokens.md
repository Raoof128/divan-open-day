# DIVAN visual tokens

These are directional defaults. Reuse existing project tokens where they already encode the same intent.

## Core palette

- Lapis night: `#16284A`
- Deep lapis: `#0D1830`
- Manuscript gold: `#C8A55A`
- Quiet gold: `#9D824A`
- Pomegranate: `#8D2335`
- Warm parchment: `#F0E4CC`
- Aged parchment: `#D8C6A5`
- Charcoal ink: `#23211E`
- Brass light: `#D7A557`

## Principles

- Gold is an accent, not a flood fill.
- Use pomegranate for small moments of emotional emphasis.
- Keep long-form poem text high contrast.
- Use subtle paper grain at low opacity and never beneath small text if it reduces legibility.
- Keep glow broad, dim, and warm. Avoid hard neon halos.

## Spacing and geometry

- Use the existing project spacing scale.
- Prefer manuscript-inspired rectangular framing with controlled corner ornament.
- Avoid introducing a second unrelated card system.
- Keep interactive controls visually modern and unambiguous even when the surrounding scene is historical.
